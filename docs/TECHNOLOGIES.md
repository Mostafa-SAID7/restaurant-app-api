# 🛠️ Technology Stack

The **Restaurant API** is built using a modern, high-performance stack centered around the .NET ecosystem, optimized for the **Apptunix** technical task requirements.

## 🔙 Backend Architecture

### Core Framework
- **.NET 8.0 SDK**: The latest long-term support (LTS) release for cross-platform excellence.
- **ASP.NET Core Web API**: High-performance, minimalist API structure.
- **C# 12**: Utilizing modern language features like primary constructors and collection expressions.

### Data & Persistence
- **Entity Framework Core 8.x**: Advanced ORM for reliable data mapping and migrations.
- **SQL Server / PostgreSQL**: Compatible with enterprise-grade relational databases.
- **Repository & Unit of Work**: Strict architectural patterns to ensure data integrity and testability.

### API Standards & Tooling
- **Swagger / OpenAPI**: Integrated interactive documentation for rapid developer onboarding.
- **AutoMapper**: Efficient object-to-object mapping (DTOs to Entities).
- **Newtonsoft.Json**: Robust JSON serialization for complex data structures.

---

## 🎨 Premium Cinematic UI

The frontend layer is a custom-engineered static interface designed to provide a "Cinematic" user experience, served directly from the API's `wwwroot`.

- **Modern HTML5**: Semantic structure for SEO and accessibility.
- **Vanilla CSS3**: 
    - **Glassmorphism**: High-end frosted glass effects using `backdrop-filter`.
    - **Cinematic Gradients**: Custom crimson-to-black aesthetic.
    - **Hover Animations**: Smooth, hardware-accelerated transitions via `transform` and `opacity`.
- **Google Fonts**: Utilizing **Inter** and **Outfit** for clean, premium typography.
- **FontAwesome**: High-quality vector icons for all UI elements.

---

## 🚀 DevOps & Infrastructure

- **Docker & Docker Compose**: Full containerization support for "it works on my machine" reliability.
- **GitHub Actions**: Automated CI/CD pipelines for build verification and deployment.
- **Multi-stage Builds**: Optimized Docker images for production environments.

---

## 🔒 Security Framework

- **Custom API Key Auth**: Secure, header-based authorization for all protected resources.
- **CORS Policies**: Strict cross-origin resource sharing to prevent unauthorized access.
- **Data Validation**: Fluent-style validation patterns for all incoming requests.
